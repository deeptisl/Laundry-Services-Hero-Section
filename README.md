# Laundry Service Hero Project

This is a simple HTML and CSS project for a laundry service hero section.

## How to Run

Just open `index.html` in your web browser. No server needed.

## What I Learned

- Using flexbox for layouts like the navbar.
- CSS Grid for the hero section.
- Viewport units (vh) to make the hero fit in one screen.
- Responsive design with media queries.
- Basic styling with colors and spacing.

## Notes

I tried different approaches:
- First used flexbox for everything, but switched to grid for the hero.
- Experimented with font sizes to fit in viewport.
- Added simple responsive design for mobile.
